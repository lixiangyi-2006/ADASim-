# ADASim – 轻量级自动驾驶仿真平台
## 项目简介
**ADASim** 是一个基于 C++/Qt 与 Python 的自动驾驶仿真平台，采用算绘分离架构。  
本项目在原作基础上，进行系统级优化与重构，重点提升仿真系统的**实时性、内存效率、可观测性、健壮性**，使其能够高效运行在资源受限的 ARM 设备上。
## 项目使用运行方法
### 环境要求
- 操作系统：Ubuntu 20.04 / 22.04（或其他支持 Qt 的 Linux 发行版）
- 编译器：g++ 9+（支持 C++17）
- Qt 版本：5.15 或 6.x（Core, Gui, Widgets, Network, Sql）
- Python 3（运行规控节点时需要）
## 调试与性能分析
### 本项目开发中系统使用了以下工具：
GDB：断点调试、多线程栈回溯
Valgrind (memcheck)：检测内存泄漏、非法访问
Callgrind ：性能热点定位（发现聚类算法 O(N²) 瓶颈）
perf：用户态 CPU 事件采样
